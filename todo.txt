SecureOps Lite — 2-Week Build Plan

Goal: build a simple but complete cybersecurity-themed project using Angular, Spring Boot, PostgreSQL, Docker, and a Go risk-scoring service, with security shifted left from the start.

Goal: Create the project foundation and keep the architecture simple.

User story: As a developer, I want a clean structure so I can build and test each part without confusion.

Architecture decision: Use Spring Boot as the main backend. Angular talks only to Spring Boot. Spring Boot talks to PostgreSQL and the Go risk service.
